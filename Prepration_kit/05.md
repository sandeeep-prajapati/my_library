### ‚úÖ **Using `LIMIT` and `OFFSET` to Paginate Results in MySQL**

`LIMIT` and `OFFSET` are used together to fetch a specific chunk of rows (like a page) from a result set.

---

### üîπ **Syntax**:

```sql
SELECT column1, column2
FROM table_name
ORDER BY column_name
LIMIT limit_value OFFSET offset_value;
```

---

### üîç **Example: Basic Pagination**

```sql
SELECT id, name
FROM users
ORDER BY id
LIMIT 10 OFFSET 0;
```

> ‚úÖ Fetches the **first 10 rows** (Page 1)

---

### üîÅ **Paginated Pages Example**:

| Page | Query                      |
| ---- | -------------------------- |
| 1    | `LIMIT 10 OFFSET 0`        |
| 2    | `LIMIT 10 OFFSET 10`       |
| 3    | `LIMIT 10 OFFSET 20`       |
| N    | `LIMIT 10 OFFSET (N-1)*10` |

---

### üîß **Formula**:

```sql
OFFSET = (PageNumber - 1) * PageSize
```
