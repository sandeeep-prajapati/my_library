---

## ğŸ“Œ **2.1 Define a Struct for JSON**

---

## âœ… **2.2 Convert Struct â†’ JSON**

```go

    panic(err)
}

```

---

## âœ… **2.3 Read JSON File â†’ Struct**

```go

    panic(err)

if err != nil {
    panic(err)
}

```

---


# ğŸ“Š **3. CSV Handling in Go**

Uses `encoding/csv`.

---

## ğŸ¯ **3.1 Write CSV File**

```go

    panic(err)
}
```

---

## ğŸ¯ **3.2 Read Entire CSV File**

```go

    panic(err)
}
defer file.Close()

reader := csv.NewReader(file)

if err != nil {
    panic(err)
}

for _, record := range records {
    fmt.Println(record)
}
```

---

## ğŸ¯ **3.3 Read CSV Line-by-Line**

```go

defer file.Close()

reader := csv.NewReader(file)

for {
    row, err := reader.Read()
    if err == io.EOF {
        break
    }
    if err != nil {
        panic(err)
    }

    fmt.Println("Name:", row[0], "Email:", row[1], "Age:", row[2])
}
```

# ğŸ’¡ Bonus: Convert CSV â†’ Struct Slice

```go
    Name  string
    Email string
    Age   int
}

func ReadCSVToStruct() []Person {
    file, _ := os.Open("people.csv")
    defer file.Close()

    reader := csv.NewReader(file)
    rows, _ := reader.ReadAll()

    persons := []Person{}

    for _, row := range rows[1:] { // skip header
        age, _ := strconv.Atoi(row[2])

        persons = append(persons, Person{
            Name:  row[0],
            Email: row[1],
            Age:   age,
        })
    }
    return persons
}
```
